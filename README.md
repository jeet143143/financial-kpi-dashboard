# Financial KPI Dashboard

A premium, executive-ready Financial KPI Dashboard designed for consulting presentations and stakeholder reviews.

## Highlights

- Glassmorphism-inspired UI with dark and light modes
- Interactive KPI cards with animated trend states
- Revenue, profit, margin, growth, and forecast storytelling
- Year-to-month drilldown with global filters
- Region and category performance breakdowns
- AI-style executive insights panel

## Run locally

From this directory, start a static server:

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173/index.html`.
